# PacketView 需求文档

## 1. 项目概述

基于 PySide6 的交互式可视化工具，在圆形区域内显示缺陷数据点、事件区域、螺旋线轨迹。前后端分离，谷歌命名规范。

## 2. 数据结构

- `Defect` / `Event` / `PacketRawMeta` / `PacketImage` — 大驼峰，定义于 `backend/models.py`
- 数组命名：`defect_array` / `event_array` / `packet_raw_meta_array`

## 3. 数据加载

- **Load Data** 按钮 → 选择文件夹，加载 `defects.csv` 和 `packet_raw_meta.csv`
- `events.csv` 懒加载：仅在右键 → View Events 时加载
- Event 指针字段（`this_ptr`, `parent`, `next`, `prev`, `next_track`, `prev_track`, `track_root`）为 event_array 下标，-1 表示空
- 加载模块位于 `backend/data_load/`

## 4. 坐标系

| 参数 | 范围 | 说明 |
| --- | --- | --- |
| 圆心 | (0, 0) | 场景坐标原点 |
| 半径 | 150000 μm | 黑色圆圈，无填充 |
| wenc | 0 ~ 262144 | 顺时针，0° 起点即 3 点钟方向 |
| xenc | 0 ~ 187500 | 外圈 → 圆心，xenc=2400 对应 r≈150000 |
| 转换公式 | `x = r·cos(θ), y = r·sin(θ)` | `θ = 2π·wenc/262144, r = 150000·(187500-xenc)/(187500-2400)` |

## 5. 螺旋线

- 依据 `packet_raw_meta_array` 逐行绘制 `(xenc_outer, wenc_left)` → `(xenc_inner, wenc_right)` 直线段
- 所有线段首尾相连形成完整螺旋轨迹
- 颜色：浅灰 `#c0c0c0`，宽度恒定（cosmetic pen）

## 6. Defect 点显示

- 位置：`wenc_xenc_to_xy(dft.w_encoder, dft.x_encoder)`
- 大小：屏幕固定约 5px，通过 `ItemIgnoresTransformations` 实现
- 颜色：正常红色 `#dc3545`，悬浮橙红，选中蓝色 `#2563a0`

### 6.1 交互

| 操作 | 行为 |
| --- | --- |
| 左键点击 defect 附近 | 自动选中最近 defect（自适应 25px 屏幕距离） |
| 左键点击空白 | 无效果 |
| 左键点击同一 defect | 取消选中 |
| 右键点击 defect 附近 | 先选中，再弹出 "View Events" 菜单 |

## 7. Defect Info 面板

- 位置：右侧栏，灰色标题栏 `#d8d6d2`
- 搜索栏：下拉选择字段 + 输入值 → Search → 画布居中并选中
- 属性列表：从 `defect_id` 起全部属性，10px 字体紧凑排列

## 8. Event 区域显示

- 右键 → View Events：懒加载 events.csv，遍历 `next` 指针链直到 -1
- 绘制两层：
  1. Defect 自身区域：红色虚线 `#dc3545`，无填充
  2. Event 链区域：浅蓝实线 `#5ba0d0`，半透明填充
- 四角多边形：由 `(xenc_outer, wenc_left)`, `(xenc_outer, wenc_right)`, `(xenc_inner, wenc_right)`, `(xenc_inner, wenc_left)` 连接
- 同一 defect 不重复绘制；Clear All Events 清除

## 9. Event Info 面板

- 位置：右侧栏 Defect Info 下方
- 标题栏：深蓝 `#b8cfe0`，内容区：浅蓝 `#e8f0f8`
- 触发：点击画布上任意 event 区域
- 显示：Event 全部 44 个属性，× 按钮关闭
- 点击 defect 时自动隐藏
- 选中 event 高亮（深蓝粗边框）

## 10. 模式栏

画布左上角三个切换按钮（34×24px，互斥，Segoe UI Symbol 字体）：

| 图标 | 功能 | 说明 |
| --- | --- | --- |
| `⌂` | Home | 拟合圆圈至视图，随后切回 Pan |
| `✋` | Pan | 拖拽平移（默认） |
| `▭` | Zoom Rect | 拖拽矩形框 → 放大至 1/2 视口 |

## 11. 画布叠加

- **右上角**：实时坐标 + 自适应比例尺
- **背景右键菜单**：Clear All Events / Clear All Packet Images
- **滚轮缩放**：×1.15，鼠标位置锚点

## 12. UI 主题

柔和暖色系：底色 `#f5f4f1`，组件 `#eeedea`，边框 `#d8d6d2`，强调色 `#2563a0`
